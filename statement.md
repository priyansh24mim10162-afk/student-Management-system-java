# Problem Statement

The VITyarthi Student Management Mini System is designed to store and manage
basic details of students. The software allows the user to add new students,
view all existing students, and search for students using their ID.

## Objective
To build a simple Java application demonstrating the concepts of OOP, modular
programming, user input handling, and data management.

## Scope
The system performs only basic operations. It does not use files or databases,
keeping the project simple and easy for beginners.

## Modules
1. Student Model – defines structure of a student
2. Student Service – handles add, search, list operations
3. Main App – user interface (console menu)
